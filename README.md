<h1>Insurance Customer Segmentation – Multi-Tool Dashboards</h1>

<p>
This project presents <b>three customer segmentation dashboards</b> created from the <b>same insurance dataset</b> using
<b>Power BI</b>, <b>Microsoft Excel</b>, and <b>Tableau</b>.  
The goal is to demonstrate how the same data can be transformed into meaningful business insights
using different analytics and visualization tools.
</p>

<hr>

<h2>📌 Project Objective</h2>
<ul>
  <li>Segment insurance customers based on demographics, policy type, income, and behavior</li>
  <li>Analyze revenue contribution by customer groups</li>
  <li>Study purchase patterns and premium trends</li>
  <li>Compare dashboard design and analytical capabilities across Power BI, Excel, and Tableau</li>
</ul>

<hr>

<h2>🛠 Tools Used</h2>
<ul>
  <li><b>Power BI</b> – Interactive data modeling and visualization</li>
  <li><b>Microsoft Excel</b> – Pivot charts, slicers, and dashboard design</li>
  <li><b>Tableau</b> – Advanced visual analytics and storytelling</li>
</ul>

<hr>

<h2>📊 Dashboards Overview</h2>

<h2>🔍 Key Insights Derived</h2>
<ul>
  <li>Business and Group policies contribute the highest revenue</li>
  <li>Middle-aged customers form the largest customer base</li>
  <li>Email and phone are the most preferred communication channels</li>
  <li>Certain occupations show higher premium purchase behavior</li>
</ul>
